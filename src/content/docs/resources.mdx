> *The curriculum is encrypted. The reading list isn't.*

A short, curated list of the most useful Zcash resources. Each link is
something you'll come back to.

## Official starting points

- **[z.cash](https://z.cash/)**: the user-facing home. Wallets, FAQs
  what-is-Zcash explainers.
- **[zcash.org](https://zcash.org/)**: the broader ecosystem hub
  maintained by the community.
- **[electriccoin.co](https://electriccoin.co/)**: Electric Coin
  Company. Blog posts, Zashi, technical announcements.
- **[zfnd.org](https://zfnd.org/)**: Zcash Foundation. zebrad
  ecosystem grants, conference, governance.
- **[zcashcommunitygrants.org](https://zcashcommunitygrants.org/)**
  Zcash Community Grants. Apply for funding, see what's been funded.

## The spec: ZIPs, and source

- **[zips.z.cash](https://zips.z.cash/)**: Zcash Improvement
  Proposals. The source of truth for the protocol.
- **[Zcash Protocol Specification (PDF)](https://zips.z.cash/protocol/protocol.pdf)**
the formal spec. Long but authoritative.
- **[github.com/zcash/zcash](https://github.com/zcash/zcash)**
  zcashd, the original full-node implementation.
- **[github.com/ZcashFoundation/zebra](https://github.com/ZcashFoundation/zebra)**
zebrad, the independent Rust full-node implementation.
- **[github.com/zcash/halo2](https://github.com/zcash/halo2)**: Halo 2
  reference implementation.

## Wallets

- **Zashi → [Zodl: Zcash Wallet](https://apps.apple.com/app/zashi-zcash-wallet/id1672392439)**:
  modern shielded-by-default mobile wallet. Originally Zashi (ECC);
  the team spun out as ZODL and renamed the app in Jan 2026. See
  [Wallets](/using/wallets/) and [/start](/start/).
- **[Ywallet](https://ywallet.app/)**: multi-pool, power-user wallet.
- **[Nighthawk](https://nighthawkwallet.com/)**: community-built
  shielded-first mobile wallet.
- **[Zecwallet Lite](https://www.zecwallet.co/)**: long-standing
  desktop wallet (Sapling-only legacy).
- **[Zcash Foundation wallet directory](https://zfnd.org/wallets/)**
  vetted wallet list.

## Merchant tooling

- **[CipherPay](https://cipherpay.app)**: non-custodial "Stripe for
  Zcash," hosted checkout, Shopify/WooCommerce plugins.
- **[ZGo](https://zgo.cash)**: in-person POS, any phone becomes a
  terminal. $1/day or $6/week.
- **[BTCPay Server](https://btcpayserver.org/)**: open-source self-
  hosted gateway with Zcash plugin.
- **[NOWPayments](https://nowpayments.io/)** / **[CoinGate](https://coingate.com/)** /
 **[Plisio](https://plisio.net/)**: custodial multi-coin gateways.

See the full lesson: [Accepting Zcash in Your Business](/using/merchants/).

## Academic and deep-technical reading

- **[Zerocash Paper (PDF)](http://zerocash-project.org/media/pdf/zerocash-extended-20140518.pdf)**
Ben-Sasson, Chiesa, Garman, Green, Miers, Tromer, Virza (2014).
  The foundational paper. Read at least the introduction.
- **Vitalik Buterin's zk-SNARK series**:
  - [Quadratic Arithmetic Programs](https://medium.com/@VitalikButerin/quadratic-arithmetic-programs-from-zero-to-hero-f6d558cea649)
  - [Elliptic Curve Pairings](https://medium.com/@VitalikButerin/exploring-elliptic-curve-pairings-c73c1864e627)
  - [zk-SNARKs Under the Hood](https://medium.com/@VitalikButerin/zk-snarks-under-the-hood-b33151a013f6)
- **[Halo paper (Bowe, Grigg, Hopwood)](https://eprint.iacr.org/2019/1021)**
recursive proof composition without trusted setup.

## Cypherpunk foundational reading

- **[A Cypherpunk's Manifesto (Eric Hughes, 1993)](https://www.activism.net/cypherpunk/manifesto.html)**
the philosophical anchor of all of this.
- **[The Crypto Anarchist Manifesto (Tim May, 1988)](https://www.activism.net/cypherpunk/crypto-anarchy.html)**
the five-years-earlier predecessor.
- The [Cypherpunk mailing list archives](https://lists.cpunks.org/)
  the actual conversations that produced all this.

## Things this site is intentionally not

- This site is not a real-time price/news source. Use exchanges or
  data dashboards for that.
- This site is not a wallet. We link to wallets; we don't host one.
- This site is not affiliated with ECC, ZF, or ZCG. It is a
  Zcash-Foundation-funded community project under the
  [Zcash Network School](https://forum.zcashcommunity.com/t/zcash-network-school/55269)
  program. We link to primary sources where they exist.

If you spot an error or want to suggest an addition, open an issue at
the GitHub repository for this site, link in the site footer.
